# Urban Mobility & Fleet Management System (Eco-Ride)

## Description

This project manages urban electric vehicle fleets across multiple hubs.
It supports vehicle management, data integrity, file handling, and analytics.

## Features

- Fleet hub management
- Vehicle creation and tracking
- Duplicate vehicle ID prevention
- JSON and CSV file operations
- Sorting and filtering
- Unit testing using Pytest

## Technologies Used

- Python
- Pytest
- VS Code
- Git & GitHub

## Project Structure

src/ → Application source code  
tests/ → Unit test cases
